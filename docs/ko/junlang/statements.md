# 문장과 블록

준랭에서 `ㅋ`은 **문장의 끝**과 **블록의 끝**을 모두 나타냅니다.

- 모든 일반 문장은 `ㅋ`으로 끝납니다.
- 블록도 `ㅋ`으로 닫습니다.

## 일반 문장

```junlang
오ㅋ
```

식 `오`(1) 하나로 이루어진 문장입니다.

## 블록

조건문, 반복문 등은 블록을 사용합니다. 블록은 여러 문장을 묶고, 마지막에 `ㅋ`으로 닫습니다.

```junlang
준서야 오 맞냐?
  오준서오ㅋ
ㅋ
```

이 코드는 다음 두 부분으로 이루어져 있습니다.

1. 블록 안의 문장: `오준서오ㅋ` — 출력 문장이 `ㅋ`으로 끝남
2. 블록 닫기: 그다음 줄의 `ㅋ`

## 붙여 쓰기

문장의 끝 `ㅋ`과 블록 닫기 `ㅋ`이 연달아 오면 `ㅋㅋ`처럼 붙습니다.

```junlang
준서야오맞냐?오준서오ㅋㅋ
```

위 코드는 앞선 예시와 **완전히 같은 의미**입니다. 줄바꿈과 들여쓰기는 가독성을 위한 것일 뿐, 실제 동작에는 영향을 주지 않습니다.
