# 주석

주석은 코드의 동작에 영향을 주지 않고 설명이나 메모를 남길 때 사용합니다. 준랭은 두 가지 주석 문법을 제공합니다.

## 한 줄 주석

`걍`부터 줄 끝까지가 주석으로 처리됩니다.

```junlang
걍 이 줄은 무시됩니다
오~준서!ㅋ 걍 변수1에 1을 대입
```

`걍` 앞에 코드가 있어도 됩니다. 그 줄에서 `걍` 이후만 주석이 됩니다.

## 블록 주석

여러 줄에 걸쳐 주석을 남기고 싶을 때 사용합니다. `참고로`로 시작하고 `알았냐?`로 끝납니다.

```junlang
참고로
이 코드는 10부터 1까지 출력합니다.
변수1을 카운터로 사용합니다.
알았냐?

오 오?~준서!ㅋ
준서야 ! 또처먹냐?
  오준서!ㅋ
  !~?오~준서!ㅋ
ㅋ
```

`참고로`와 `알았냐?` 사이의 모든 내용은 무시됩니다.
