# 에러 메세지

준랭은 실행 중 문제가 발생하면 정해진 에러 메시지를 출력합니다.

## 에러 메시지 형식

에러는 다음과 같은 형식으로 출력됩니다.

```text
[줄]:[칸]: [에러 메시지]
```

- **줄**: 에러가 발생한 줄 번호
- **칸**: 해당 줄에서 에러가 발생한 글자 위치
- **에러 메시지**: 정해진 문구 중 하나

### 예시

```text
3:5: 이건 기하반도 안 하는 실수인데?
```

3번째 줄 5번째 칸에서 0으로 나누기가 발생했음을 의미합니다.

## 에러 목록

| 상황 | 에러 메시지 |
| --- | --- |
| 실행할 파일을 못 찾거나 못 읽음 | `아 먹을거 어디있어` |
| 잘못된 입력 표기 또는 잘못된 문법 | `어! 이건 오~준서! 도 안 할 실수인데?` |
| 입력 함수에서 입력이 들어오지 않음 | `준서가 먹을거 없어서 슬프대` |
| 0으로 나누기 | `이건 기하반도 안 하는 실수인데?` |
| 제곱 지수가 소수라서 처리할 수 없음 | `준서가 어렵대` |
| 자릿수 초과 (`오`가 10개 이상 연속) | `오 10개? 좀 진정해` |
| `ㅋ` 마침표 누락 | `준서야 그만 처 먹어` |
| 정의되지 않은 변수 참조 | `그 준서는 누구야?` |
| 루프 밖에서 `더처먹어` / `그만처먹어` 사용 | `먹지도 않았는데 뭘 그만 처 먹어?` |
| 닫을 블록 없는 `ㅋ` | `넌 이 상황이 처 웃겨?` |

::: tip
"실행할 파일을 못 찾거나 못 읽음" 에러는 줄/칸 정보 없이 출력됩니다.
:::
