# 입출력
이 문서에서는 입력과 출력에 대해 다룹니다.

## 입력

표준 입력에서 한 줄을 읽어 수를 가져옵니다.

```junlang
오ㅋ준ㅋ서ㅋ
```

### 입력값 표기

입력은 [수 체계](./number-system)를 따릅니다.

| 입력 | 값 |
| --- | --- |
| `오오 오?` | 20 |
| `?오오오` | -3 |
| `오ㅋ오오오오오` | 1.5 |

### 예시

```junlang
오ㅋ준ㅋ서ㅋ~준서!ㅋ
```
입력값을 `!`에 대입합니다.

```junlang
오준서오ㅋ준ㅋ서ㅋㅋ
```
입력된 값을 그대로 출력합니다.

```junlang
오준서오ㅋ준ㅋ서ㅋ~오ㅋ
```
입력값에 `오`(1)을 더하고, 출력합니다.

## 출력

```junlang
오준서[식]ㅋ
```

### 예시

```junlang
오준서오~오ㅋ
```
`오`(1) + `오`(1) 값을 출력합니다. (출력값: `오오`)

```junlang
오준서!ㅋ
```
`!` 의 값을 출력합니다.

```junlang
오준서오오오#오오ㅋ
```
`오오오`(3) 나누기 `오오`(2)의 값을 출력합니다.

### 출력 형식

출력값은 준랭 숫자 표기로 변환되어 나옵니다.

- **정수**는 정수 표기로 출력됩니다.
- **유한소수**를 출력할때 마지막 자리의 `오?`(0) 은 생략합니다. (예: `오ㅋ오오 오?` -> `오ㅋ오오`)
- **무한소수** 또는 **소수점 이하 8자리를 초과하는 값**은 소수점 이하 8자리에서 반올림됩니다.

| 코드 | 출력 |
| --- | --- |
| `오준서오 오?#오오ㅋ` | `오오오오오` |
| `오준서오오오#오오ㅋ` | `오ㅋ오오오오오` |
| `오준서오오 오?#오 오? 오?ㅋ` | `오?ㅋ오오` |
