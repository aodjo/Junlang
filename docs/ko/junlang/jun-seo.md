# 준, 서

`준`과 `서`는 식의 범위를 묶을 때 사용됩니다. 일반적인 프로그래밍 언어의 여는/닫는 기호 한 쌍과 같은 역할을 합니다.

| 준랭 | 일반적인 표기 |
| --- | --- |
| `준` | `(` |
| `서` | `)` |

## 우선순위 바꾸기

연산자 우선순위를 무시하고 특정 부분을 먼저 계산하고 싶을 때 사용합니다.

```junlang
준오~오서.오오
```

`(1 + 1) × 2` -> **`오오오오`(4)**

`준`, `서`가 없었다면 곱셈이 먼저 계산되어 `1 + (1 × 2) = 3`이 되었겠지만, `준`, `서`로 묶었기 때문에 덧셈이 먼저 계산됩니다.

```junlang
준준오~오서.오오오서.오오
```

`((1 + 1) × 3) × 2` → **`오 오오`(12)**

`준`, `서`는 중첩해서 사용할 수 있습니다.

## 변수의 범위 명시

`!`(변수 이름) + `@`(같나)가 붙어 있을 때, 인터프리터는 [붙여 쓰기 규칙](./parsing-rules)에 따라 해석합니다. 이때 의도가 모호해지면 `준`, `서`로 변수의 범위를 명확히 잘라줄 수 있습니다.

```junlang
준!!서@오오
```

`변수 2번 == 2`로 해석됩니다. 자세한 내용은 [붙여 쓰기 규칙](./parsing-rules)을 참고하세요.

::: tip
`준`, `서`는 항상 쌍으로 사용해야 합니다. 한쪽만 쓰면 문법 에러가 발생합니다.
:::
