# 반복문

반복문은 조건이 참인 동안 블록 안의 문장들을 계속 실행합니다.

```junlang
준서야 [조건] 또처먹냐?
  [문장들]
ㅋ
```

조건이 거짓이 되면 반복이 멈추고 블록 다음 문장으로 넘어갑니다.

::: tip
조건이 어떻게 참/거짓으로 평가되는지는 [참과 거짓](./boolen)을 참고하세요.
:::

## 예시

1부터 10까지 출력하는 코드입니다.

```junlang
오 오?~준서!ㅋ
준서야 ! 또처먹냐?
  오준서!ㅋ
  !~?오~준서!ㅋ
ㅋ
```

이 코드는 다음 단계로 동작합니다.

1. 변수1에 `오 오?`(10)을 대입합니다.
2. 변수1이 0이 아니면 (= 참이면) 블록 안을 실행합니다.
   1. 변수1의 값을 출력합니다.
   2. 변수1에 `?오`(-1)을 더한 값을 다시 변수1에 저장합니다. (= 1만큼 감소)
3. 변수1이 0이 되면 반복이 멈춥니다.

결과적으로 `10, 9, 8, ..., 1`이 출력됩니다.

::: warning 무한 루프에 주의하세요
조건이 영원히 참인 상태로 남으면 반복이 끝나지 않습니다. 블록 안에서 조건에 영향을 주는 값(주로 변수)을 반드시 변경해야 합니다.

```junlang
준서야 오 또처먹냐?
  오준서오ㅋ
ㅋ
```

위 코드는 조건이 항상 `오`(1)이므로 영원히 멈추지 않습니다.
:::

## 반복 중간에 빠져나가기

반복문 안에서 [흐름 제어](./flow-control) 키워드 `더처먹어`(continue)와 `그만처먹어`(break)를 사용해 반복의 흐름을 바꿀 수 있습니다.
